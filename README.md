# MailAssistant

Инструмент для работы с почтой через AI-агента. Позволяет читать письма, суммаризировать их и сохранять черновики — всё через чат с ИИ, без раскрытия данных третьим сервисам.

---

## Установка

### 1. Установите зависимости

```bash
npm i
```

### 2. Настройте переменные окружения

Скопируйте содержимое файла `.env.example` в новый файл `.env`:

```bash
cp .env.example .env
```

Затем откройте `.env` и заполните поля:

```env
EMAIL_USER=your_email@gmail.com
EMAIL_PASSWORD="xxxx xxxx xxxx xxxx"
```

> **Важно:** `EMAIL_PASSWORD` — это **не пароль от вашей почты**.  
> Это специальный **пароль для приложения**, который создаётся отдельно в настройках аккаунта Google.

#### Как создать пароль для приложения (Google):

1. Перейдите в [myaccount.google.com](https://myaccount.google.com)
2. Выберите **Безопасность** → **Двухэтапная аутентификация** (должна быть включена)
3. Прокрутите вниз до раздела **Пароли приложений**
4. Создайте новый пароль для приложения с названием, например, `MailAssistant`
5. Скопируйте сгенерированный 16-значный пароль в `.env`

---

### 3. Запустите сервер

```bash
npm start
```

Сервер запустится на `http://localhost:3000`.

---

## Безопасность

Файл `.env` **не покидает пределы вашего компьютера** — в проекте настроен `.gitignore`, который исключает его из системы контроля версий. Ваши учётные данные никогда не будут загружены в репозиторий.

## Как пользоваться

Взаимодействие с почтой происходит **через чат с AI-агентом** (например, Google Antigravity):

| Что вы хотите сделать | Что написать агенту |
|---|---|
| Загрузить письма | «Загрузи мои последние письма» |
| Прочитать письма | «Покажи мне письма» |
| Суммаризировать | «Суммаризируй мои письма» |
| Написать ответ | «Напиши деловой ответ на письмо от X» |
| Сохранить черновик | «Сохрани это письмо в черновики» |
| Отправить письмо | «Отправь письмо на адрес X» |

Агент использует локальный сервер для чтения почты через **IMAP** и отправки через **SMTP**. Анализ и генерация текста выполняются самим AI-агентом — без передачи данных в OpenAI или любые внешние сервисы.

## API эндпоинты (для ручного использования)

| Метод | Путь | Описание |
|---|---|---|
| `GET` | `/api/health` | Проверка состояния сервера |
| `GET` | `/api/emails?limit=20` | Загрузить письма с почты (IMAP) |
| `GET` | `/api/emails/cached` | Получить кэшированные письма |
| `GET` | `/api/emails/:id` | Получить одно письмо по ID |
| `POST` | `/api/send` | Отправить письмо |
| `POST` | `/api/draft` | Сохранить черновик |
